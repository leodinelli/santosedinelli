# Planejamento do Site de Ar Condicionado

## Análise das Referências

Baseado na análise dos principais sites de ar condicionado no Brasil (CentralAr, Dufrio, Polo Ar), identifiquei os seguintes elementos essenciais:

### Elementos Comuns dos Concorrentes:
- **Header com busca prominente** - Campo de busca centralizado
- **Categorização por BTUs** - Calculadora de BTUs para orientar o cliente
- **Categorização por tipo** - Split, Inverter, Janela, Cassete, Multi Split
- **Categorização por ambiente** - Quarto, Sala, Área Gourmet
- **Ofertas em destaque** - Banners promocionais e produtos em oferta
- **Informações de parcelamento** - Destaque para condições de pagamento
- **Calculadora de BTUs** - Ferramenta para dimensionamento
- **Atendimento via WhatsApp** - Chat integrado
- **Cores predominantes** - Azul, branco e verde (cores que remetem ao frio)

## Conceito de Design

### Identidade Visual
- **Paleta de cores:**
  - Azul principal: #1E40AF (transmite confiança e frescor)
  - Azul secundário: #3B82F6 (para elementos interativos)
  - Verde accent: #10B981 (para CTAs e ofertas)
  - Cinza neutro: #6B7280 (para textos)
  - Branco: #FFFFFF (background principal)

### Tipografia
- **Fonte principal:** Inter (moderna, legível, profissional)
- **Hierarquia:** H1 (32px), H2 (24px), H3 (20px), Body (16px)

### Estilo Visual
- **Design moderno e clean** com muito espaço em branco
- **Cards com sombras suaves** para produtos e serviços
- **Gradientes sutis** em backgrounds de seções
- **Ícones minimalistas** para categorias e funcionalidades
- **Imagens de alta qualidade** de produtos e ambientes climatizados

## Estrutura do Site

### 1. Header
- Logo da empresa
- Menu de navegação (Produtos, Serviços, Sobre, Contato)
- Campo de busca
- Ícones: Conta do usuário, Carrinho, WhatsApp

### 2. Hero Section
- Banner principal com oferta especial
- Calculadora de BTUs em destaque
- CTA principal "Encontre seu Ar Condicionado"

### 3. Categorias de Produtos
- **Por Tipo:** Split Hi Wall, Inverter, Janela, Cassete, Multi Split, Piso Teto
- **Por BTUs:** 9.000, 12.000, 18.000, 24.000, 30.000, 36.000+
- **Por Ambiente:** Quarto, Sala, Escritório, Loja, Área Gourmet

### 4. Produtos em Destaque
- Grid responsivo de produtos
- Cards com imagem, nome, preço, desconto
- Filtros por marca, preço, BTUs, tipo

### 5. Serviços
- **Instalação:** Equipe técnica especializada
- **Manutenção:** Limpeza e revisão preventiva
- **Assistência Técnica:** Reparo e troca de peças
- **Consultoria:** Dimensionamento e projeto

### 6. Diferenciais
- Entrega rápida
- Instalação profissional
- Garantia estendida
- Parcelamento sem juros
- Atendimento especializado

### 7. Footer
- Links úteis
- Informações de contato
- Redes sociais
- Certificações e selos

## Funcionalidades Técnicas

### E-commerce
- **Catálogo de produtos** com filtros avançados
- **Carrinho de compras** persistente
- **Calculadora de frete** por CEP
- **Simulador de parcelas**
- **Comparador de produtos**
- **Lista de desejos**

### Ferramentas Especiais
- **Calculadora de BTUs** - Baseada em área do ambiente
- **Simulador de economia** - Comparação Inverter vs On/Off
- **Agendamento de serviços** - Formulário para instalação/manutenção
- **Chat online** - Integração com WhatsApp

### Responsividade
- **Mobile First** - Otimizado para dispositivos móveis
- **Breakpoints:** 320px, 768px, 1024px, 1440px
- **Touch-friendly** - Botões e elementos adequados para toque

## Páginas Principais

1. **Home** - Apresentação geral e produtos em destaque
2. **Produtos** - Catálogo completo com filtros
3. **Produto Individual** - Detalhes, especificações, compra
4. **Serviços** - Instalação, manutenção, assistência
5. **Calculadora BTUs** - Ferramenta de dimensionamento
6. **Carrinho** - Resumo da compra e checkout
7. **Sobre** - História da empresa e diferenciais
8. **Contato** - Informações e formulário

## Tecnologias

- **Frontend:** React 18 com TypeScript
- **Styling:** Tailwind CSS
- **Ícones:** Lucide React
- **Animações:** Framer Motion
- **Formulários:** React Hook Form
- **Estado:** Context API
- **Build:** Vite
